# 个人轻量级全能工作台 (Geek Workspace) PRD v2.0

本项目是一个专注于个人极客生产力的桌面端工作台，采用前后端分离架构本地部署，旨在提供“沉浸、极速、无干扰”的任务与知识管理体验。

## 🛠 技术栈基建 (Tech Stack)
* **前端**: Vue 3 + TailwindCSS (实现极简且高级的 UI 视觉效果)
* **后端**: Python / FastAPI (高并发、异步底层驱动)
* **数据库**: SQLite (单文件存储) + SQLAlchemy ORM
* **桌面化**: PyWebView (极低内存占用的原生视窗)

## ✨ 核心业务模块 (Core Features)

### 1. 闪电记事本 (Flash Notebook)
* **Markdown 原生支持**：放弃老旧的富文本，支持全语法 Markdown 渲染，所见即所得，支持代码高亮与快捷键排版。
* **全局全文检索引擎**：支持 `Ctrl+K` 唤出全局搜索框，毫秒级在成百上千篇笔记的内容中高亮匹配关键字。
* **附件与图床**：支持将本地图片直接拖拽进笔记，后端自动将其转换为本地 Base64 或本地文件链接存储。
* **数据防灾机制**：包含“回收站”功能（软删除，30天内可恢复）及“一键导出所有笔记为 Markdown/PDF”功能。

### 2. 敏捷任务看板 (Agile Todo Board)
* **多视图切换**：支持基础的“列表视图”，以及可拖拽的“看板视图 (Kanban)”（如：待处理、进行中、已完成），任务状态随拖拽实时更新。
* **父子任务嵌套**：支持为主任务建立无限层级的“子任务 (Sub-tasks)”，并自动计算主任务的完成进度条 (0% - 100%)。
* **番茄工作法 (Pomodoro)**：内置 25 分钟番茄钟倒计时，开启后界面进入沉浸模式（隐藏侧边栏）。

### 3. 智能提醒与系统交互 (System Integration)
* **全局快捷键呼出 (Global Hotkeys)**：即使软件在后台隐藏，按下 `Alt + Space` 即可在桌面中央弹出一个极简的“灵感捕捉框”，敲击回车瞬间保存，不打断当前工作流。
* **智能推迟提醒**：当桌面右下角弹出任务通知时，支持操作“推迟 10 分钟”、“推迟到明天”等智能化选项。

### 4. 外观与个性化 (UI/UX Settings)
* **自适应主题**：支持“深色模式 (Dark Mode)”与“浅色模式”，可跟随 Windows/Mac 系统的昼夜主题自动切换。
* **数据面板 (Dashboard)**：首页提供数据图表（如：本周专注时间统计、笔记产出热力图），给予用户正向心理反馈。
