# 个人轻量级记事与计划清单 (Personal Notebook & Todo)

这是一个专注于个人生产力的轻量级桌面应用程序，集成了记事本、任务清单与系统级定时提醒功能。项目采用前后端分离架构进行本地化部署与打包，确保数据绝对私密且内存占用极低。

## 🛠 技术栈 (Tech Stack)

* **前端 (Frontend)**: Vue 3 (提供流畅的现代化交互界面)
* **后端 (Backend)**: Python / FastAPI (负责核心逻辑、本地存储与定时器)
* **本地存储 (Database)**: SQLite (单文件轻量级数据库)
* **桌面端构建 (Desktop Wrapper)**: PyWebView + PyInstaller (极低内存占用的系统视窗容器)

## ✨ 核心功能模块 (Features)

### 1. 记事本模块 (Notebook)
* **富文本记录**：支持加粗、列表及基础排版，方便记录和沉淀重要信息。
* **分类与检索**：提供灵活的分类标签（如工作、生活、学习等），并支持全局关键字快速检索。
* **本地安全存储**：所有记事数据均保存在本地 SQLite 数据库中，无云端同步，保障绝对的个人隐私。

### 2. 任务清单模块 (Todo List)
* **每日计划视图**：直观展示今日待办任务，完成后自动划线并沉底，保持界面清爽。
* **优先级标记**：支持为不同任务设置颜色或星号标记，突出显示紧急/重要事项，合理规划每日精力。
* **周期性任务**：针对规律性活动（如每日打卡、每周总结），支持自动生成重复任务，减少手动录入成本。

### 3. 定时提醒模块 (Reminders)
* **精准时间设定**：可为任意独立的记事或待办任务设定精确的触发时间。
* **系统级通知**：依托 Python 后端与操作系统底层交互，触发 Windows/Mac 桌面右下角的原生消息弹窗。
* **稍后提醒 (Snooze)**：弹窗触发时支持"推迟"操作，避免在专注工作时错过重要提醒。
